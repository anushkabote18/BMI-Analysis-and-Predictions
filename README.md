
# BMI Analysis & Predictive Modeling

An end-to-end Machine Learning project that analyzes height and weight metrics to predict body categories and continuous numeric values. This project showcases data preprocessing, handling class imbalances, and deploying models for real-time inference.

## 🚀 Features & Architecture
* **Classification Model:** Maps height/weight inputs to specific health categories (e.g., Underweight, Normal) using optimized decision boundaries.
* **Regression Model:** Employs Linear Regression to calculate continuous, mathematical target weight estimates based on dataset trend lines.
* **Live Inference Demos:** Built-in interactive code blocks that accept custom user inputs (in inches/pounds) for instant model predictions.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-Learn
* **Environment:** Jupyter Notebook
